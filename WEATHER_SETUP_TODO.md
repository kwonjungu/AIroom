# 🌤️ 오늘의 날씨·경보 알리미 — 사용자 할 일 체크리스트

헤더 우상단에 **백암면(경기 용인 처인구) 날씨 + 폭염·미세먼지·기상 경보** 위젯을 추가했습니다.
코드는 모두 배포되었지만, **API 인증키 1개를 발급·등록해야** 실제 데이터가 표시됩니다.
(키가 없으면 위젯은 안전하게 "날씨 정보 없음"으로만 표시되고 앱 동작에는 영향 없음)

---

## ✅ 1단계 — data.go.kr 인증키 발급 (필수, 1회)

1. [공공데이터포털(data.go.kr)](https://www.data.go.kr) 회원가입 / 로그인
2. 아래 **2개 API "활용신청"** (개발계정, 자동 승인):
   - **기상청_기상특보 조회서비스** → 폭염·호우·강풍·대설·한파·태풍 등 경보
     - https://www.data.go.kr/data/15000415/openapi.do
   - **한국환경공단_에어코리아_대기오염정보** → 미세먼지(PM10/PM2.5)
     - https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15073861
   - (선택) **기상청_단기예보 조회서비스** → 현재기온 표시용
     - https://www.data.go.kr/data/15084084/openapi.do
3. 마이페이지 → 오픈API → **일반 인증키(Decoding)** 값 복사
   - ⚠️ **Decoding(URL-decoded 원본) 키**를 사용하세요. (서버가 내부에서 자동 인코딩함)
   - 같은 계정이면 키 **하나로 위 3개 API 공용**입니다.

## ✅ 2단계 — Vercel 환경변수 등록 (필수)

Vercel 프로젝트 → Settings → Environment Variables 에 추가:

| Key | Value |
|---|---|
| `DATA_GO_KR_KEY` | 위에서 복사한 **Decoding 인증키** |

- Production/Preview 모두 체크 → Save → **Redeploy**(재배포해야 적용됨)
- (로컬 테스트 시) 레포 루트 `.env` 파일에 `DATA_GO_KR_KEY=발급키` 한 줄 추가 (`.env`는 git 제외됨)

## ✅ 3단계 — 동작 확인 + 지역 미세조정

재배포 후 로그인한 상태에서 브라우저 주소창에:

```
https://a-iroom.vercel.app/api/weather?debug=1
```

를 열어 응답 JSON을 확인하세요. (로그인 토큰 필요 — 안 되면 그냥 메인 화면 우상단 위젯으로 확인)

확인할 점:
- [ ] `result.temp` 에 현재기온 숫자가 들어오는지
- [ ] `result.dust` 에 PM10/PM2.5 수치가 들어오는지
  - **측정소가 비어있으면** → 환경변수 `WEATHER_DUST_STATION` 값을 백암면 근처 측정소명으로 바꾸기
    (기본값 `김량장동`. 후보: 처인구/용인 지역 측정소. 에어코리아 "우리동네 대기정보"에서 백암면 최근접 측정소명 확인)
- [ ] `errors` 항목에 메시지가 있으면 → 해당 API 활용신청/키가 안 된 것 (1~2단계 재확인)
- [ ] 폭염/미세먼지 경보 시 위젯이 **주황/빨강 + 깜빡임**으로 바뀌는지 (여름 폭염 시즌에 확인)

## 🔎 참고 / 알려진 한계 (개발자 메모)

- **기상특보 지역 필터**는 통보문 텍스트에서 `용인 / 경기남부내륙 / 처인` 키워드로 백암면 해당분만 추출합니다.
  실제 폭염특보 발효 시 통보문 형식이 예상과 다르면 누락될 수 있으니, 첫 폭염 발효일에 `?debug=1`로 한 번 점검 권장.
  (키워드는 `server.js`의 `WARN_AREA_KEYWORDS`에서 조정 가능)
- **호출량**: 서버에서 10분 캐시(전 사용자 공유)하므로 에어코리아 개발계정 한도(500/일) 안에서 안전합니다.
  단, `?nocache=1`로 호출하면 캐시를 우회하니 테스트 외에는 쓰지 마세요(한도 소모).
- **측정소 기본값(`김량장동`)** 은 추정값입니다. 3단계에서 실측 후 확정해 주세요.
- 격자 좌표(백암면): `nx=67, ny=118` (위경도 37.16, 127.38 자동 변환).
